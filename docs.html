General

The foremost purpose and intention regarding Cosmic is to ensure the users in a Discord server get entertainment. That's why the main innovation of Cosmic is it's entertainment commands, and among it the Moderation capabilities.

DM Messages

Cosmic has a few commands that personally message the user, and personally messages the owner of a Discord server when Cosmic first gets added. We designed Cosmic this way so your server chats could stay nice and clean so that you could easily read what the bot had messaged you without your chat moving if you have an active server.

Command Summery

General

++Botinfo Shows information on Cosmic.
++Ping Shows bot ping to the Discord API.
++Help Shows all of Cosmic's commands.
++Rules Shows you pre-determined rules.
++Links Shows links to important sources.
++Serverinfo Grabs information on the server.
++Userinfo Grabs information on yourself or a user you tag.
++About Learn more about Cosmic.
++Uptime Shows how long Cosmic has been active.
++Reportbug Reports a bug to the developers of Cosmic.
++Suggest Allows a user to suggest a feature to the developers.

Moderation

Keep your server in order with Cosmic's moderation and management commands. All the moderation and administration commands are listed in ++Help, under the staff district.

Commands using the Manage Messages permission.
++Mute Permentantly mute a user.
++Unmute Unmutes a user.
++Reportuser - (All users can perform reports) Allows a user to report any other user for misbehaviour.
++Purge Delete a set amount of messages.
++Warn Warns a user.
++Warnings Checks how many warnings a user has.
Commands using the Kick Members and Ban Members permissions.
++Kick Will kick the user from the Discord server.
++Ban Will ban the user from the server.

Administrator Commands

Here is Cosmic's administrator commands.

++PM Personally messages a user.
++Poll Creates a poll.
++Say Allows Cosmic to say a message in the current channel.
++Acceptstaff Accept a user as staff on your server (This only PM's them)
++Demotestaff Demote a user from staff on your server (This only PM's them)

Ticket Commands

Our Ticket System is simple, yet powerful.

++New Creates a new ticket that only the user and people with the "Support Team" role can see.
++Close Closes the ticket. (There is a simple confirmation command just in case)
++Aboutticket Displays information about Cosmic's ticket system.
